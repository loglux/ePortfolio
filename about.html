About this ePortfolio

StudyHub is a learning ePortfolio showcasing HTML5, CSS3, and JavaScript. It implements DOM manipulation, forms with validation, storage APIs, canvas graphics & animation, geolocation, Web Workers, WebSockets, and API consumption with both fetch and XMLHttpRequest (plus a jQuery example). The UI is responsive using Bootstrap.

Key features

  • Responsive UI (Bootstrap grid, utilities)
  • Notes module with localStorage (CRUD, search, sorting)
  • Pomodoro timer using a Web Worker and canvas progress
  • Canvas demo: shapes, gradients, text, animation, and drawing
  • Forms page: HTML5 + custom JS validation (regex), session draft
